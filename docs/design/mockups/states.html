State specifications

StateColorMotionIconDurationTap behavior
Idle cream glass + tangerine accents 4s breathe + 24s halo rotate mic outline, tangerine until tapped start session

Transcribing cream glass + diagonal shimmer 1.1s diagonal shimmer sweep mic outline, muted ~200–500ms (Deepgram final) disabled (transitions automatically)
Thinking cream glass + tighter halo rotation 3 bouncing dots (1.2s stagger), halo rotates at 6s none (replaced by dots) ~800–1500ms (Claude TTFT) disabled (cancel returns to idle)
Speaking cream glass + audio-wave halos 5 bouncing EQ bars (0.8s), 2s outward sound waves none (replaced by EQ bars) length of buddy reply interrupt buddy, return to idle
